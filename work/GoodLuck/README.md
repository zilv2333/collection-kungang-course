## 说明
这是一个用于分析引体向上动作质量的程序，通过上传引体向上正侧面视频对动作进行分析。由于知识库部分还未搭建完成，所以效果不会很好。
<br>

## 使用
docker 启动后在80端口有一个网页可以进入使用，管理员账号admin密码123456。管理员设置了默认路由但是管理员页面暂时未完善，体验功能请自行注册一个账号或路由到/main
